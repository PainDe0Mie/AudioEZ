What's new in V1.1 🎉

AudioEZ v1.1 is here — the biggest update yet. New smart features, a cleaner interface, and a ton of fixes under the hood. Here's everything that's new 👇

✨ New features

  • Auto-Preamp — Hit "Auto" next to the preamp slider and AudioEZ figures out the ideal negative preamp for you based on your current EQ. No more clipping.
  • Arrow key control on the graph — Select a point, then use ↑↓ to adjust gain by 0.5 dB and ←→ to move its frequency. Hold Shift for finer 0.1 dB / 0.01-step moves.
  • Hover tooltip on the curve — Move your mouse over the EQ graph and you'll see the exact frequency and gain at your cursor — no need to click.
  • Wavelet import — You could already export to Wavelet (.json) format. Now you can import those files too.
  • Preset tags & filtering — Tag any preset (IEM, Over-ear, Gaming, Music, Custom…) and filter the preset list by tag. Rename button also added next to the list.
  • Band count conversion on export — Export a 10-band EQ as 5 bands (or any other count). AudioEZ resamples the curve automatically.
  • Undo / Redo — Up to 50 steps. Ctrl+Z / Ctrl+Y or the toolbar buttons.
  • A/B Compare — Freeze a reference snapshot and switch between it and your live EQ on the fly.
  • Reset EQ — One button to flatten everything back to zero.
  • Scroll wheel on graph — Hover a band point and scroll to adjust gain. Shift = fine mode.
  • Export to APO Include — Writes directly to the EQ APO config folder without touching your other plugins.
  • System Tray — AudioEZ stays in your tray. Toggle the EQ or reopen the window without keeping it on your taskbar.

🧠 Adaptive Filter improvements (Experimental)

  • Live detection badge — See what AudioEZ is currently hearing (genre, confidence %) right in the settings panel while Adaptive is on.
  • Advanced parameters — Tune speech sensitivity, music sensitivity, how long a genre needs to stay detected before switching, cooldown between switches, and transition speed — all from the UI, no restart needed.
  • Profile selector — Enable or disable specific profiles (Speech, Rock, Jazz, etc.) so the filter only switches to genres you care about.
  • Manual override — Touch any EQ slider while Adaptive is running and it pauses automatically, then resumes on its own after a timeout.
  • Much smoother transitions — Config file writes are now throttled during transitions instead of spamming your disk 50 times per second.
  • Better crash protection — Thread-safe shutdown, safe recorder teardown, and robust restore of your original EQ when you turn Adaptive off.

⚡ Performance

  • APO config file writes debounced to max ~80 ms — no more hammering disk on every pixel dragged.
  • EQ curve is only recalculated when a parameter actually changed (hash-based cache).
  • Startup verification dialog no longer freezes the UI with a blocking sleep — uses a proper timer now.

🎨 UI polish

  • Toolbar regrouped logically — Undo/Redo | A/B + Reset | Import / Save / Export | Delete (now isolated so you can't accidentally hit it).
  • Rename preset moved to a dedicated pencil button (double-click on the dropdown doesn't work in Qt — now fixed).
  • Dropdown highlights no longer stay blue after hovering.
  • Toggle switches in the Parameters modal now animate correctly.
  • Shimmer animation on the Experimental badge actually works.
  • Safe Mode — caps every band and preamp to a configurable limit (default ±12 dB).

🐛 Bug fixes

  • Fixed crash on file import (NameError: bands).
  • Fixed export crash with QFileDialog (PyQt6 enum change).
  • Fixed set_q_factor applying to the wrong object.
  • Fixed Adaptive Filter crash when the RTGD module isn't installed.
  • Fixed export modal showing "Warning: x" placeholder on open.
  • Fixed close button for Parameters modal accidentally closing Export modal.
  • Fixed backup_config / restore_config calling a method that didn't exist.
  • Fixed hardcoded 10-band limit in several places — AudioEZ now works with any band count.
  • Fixed duplicate version-check function in startup (second one was crashing silently on network errors).
  • Fixed Discord RPC default inconsistency.
  • Preset rename and delete now also clean up associated tags.
